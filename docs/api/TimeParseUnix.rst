TimeParseUnix - Преобразвание строки в Unix Time
================================================================================

.. code-block:: lua 

    --Пример

    output.ts = TimeParseUnix (

                                "07/26/2019  11:17:00  +03:00" ,

                                "01/02/2006  15:04:05  -07:00" 

                               )
    [ 
     
         [ "output" : {
 
                        "ts" : 1564129020 
 
                       }
 
         ]

    ]

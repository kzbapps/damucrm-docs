StringToJson - Преобразование строки в table
======================================================================

.. code-block:: lua

   --Пример 
   
   s = StringToJson( 
                     [
                       [
                         {"ok": 1}
                       ]
                     ]
                   )
   if s.id == nil then 
           return
           end              --В случае ошибки парсинга Json, возвращает пустой объект {}

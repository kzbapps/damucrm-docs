Regexp find string submatch - поиск подстроки.
==============================================================================

.. code-block:: lua 

   output.zapros = RegexpFindStringSubmatch(

                                            "Запрос № : 1234567 ;

                                             Регион : Петропавловск ;

                                             Luno :12345678 " ,
                                             
                                             [ [ Запрос №: 123456789\Регион:Петропавловск\Luno:12345678 , 1]  ]
